## Notas de Implementación - Backend

✅ **Clean Architecture + MVC** - Separación por capas en proyectos
✅ **SignalR** - Comunicación en tiempo real bidireccional
✅ **Inyección de Dependencias** - Service Collection de ASP.NET
✅ **InMemory Storage** - Para desarrollo y pruebas rápidas
✅ **Control de Errores** - Try-catch en Hub y UseCases
✅ **Logging** - ILogger en todos los servicios
✅ **CORS** - Configurado para React Native
✅ **JSON Protocol** - Configuración case-insensitive
✅ **Connection Management** - Tracking de conexiones por sala
✅ **Domain-Driven Design** - Entities, Value Objects, Aggregates
✅ **Singleton Scope** - Repositories y ConnectionManager
✅ **Scoped** - UseCases para cada request
✅ **Validaciones** - En entidades del dominio
✅ **Excepciones personalizadas** - DomainException y derivadas
